JavaScript :  
----------
Easy to use programing/scripting lang, which can be embedded in the header or body element of the web page.
-Performs calculations,validations, special effects, graphics etc..
- Is a scripting lang that helps in making html pages live
-invented by Brendan Eich


Limitations of javascript : cant read/write to hard disk files, and call other programs
-Does not have any multithreading or multiprocessor capabilities
